# この Runbook（手順書）について
この Runbook（手順書）では `@StoneDot` が学習した内容を整理して、システム管理作業時に参照することを意図しています。
そのため、Runbook 本体の内容は、一覧性・検索性を考慮して可能な限りフラットに配置しています。

詳細な話やブログ記事的な性質を持つものは `article` に配置します。
